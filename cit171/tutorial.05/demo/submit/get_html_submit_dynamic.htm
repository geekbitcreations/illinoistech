GET Example with Dynamic HTML Submit Button

This is an example of how to use GET HTTP Method with a dynamic HTML submit button. A possible use for this type of setup with three submit buttons would be one button for Add, one for Delete, and another for Change.

When you use GET the contents of your form will become url encoded into a query string appended after a ? as name=value pairs. For example if your form was submitted to form_processor.html and took your firstName and your lastName - the corresponding query string attached to the URL would be…..
form_processor.html?firstName=Santa&lastName=Claus

After you click the Submit button look to see how the form data was passed as a query string in the Web browser address bar.
