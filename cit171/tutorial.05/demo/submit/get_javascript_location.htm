Dynamic Query String Example with JavaScript Location Object

This is an example of how to build a dynamic query string and use it with the JavaScript location object.

The contents of your form will become url encoded into a query string appended after a ? as name=value pairs. For example if your form was submitted to form_processor.html and took your firstName and your lastName - the corresponding query string attached to the URL would be…..
form_processor.html?firstName=Santa&lastName=Claus

After you click the Submit button look to see how the form data was passed as a query string in the Web browser address bar.
